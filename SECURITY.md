# Security Policy

## Supported Versions
Only the latest version of the extension on the `main` branch is actively supported with security updates.

## Reporting a Vulnerability
Since this extension operates entirely offline and locally, the attack surface is minimal. However, if you discover a vulnerability (e.g., XSS via maliciously crafted URLs), please do not open a public issue. 
Email the maintainer directly. We will review the report and issue a patch as soon as possible.
