# Contributing to Pro Time Tracker

First off, thank you for considering contributing! 

## How to Contribute
1. **Fork the repo** and create your branch from `main`.
2. **Write clear code** that follows the existing Vanilla JS structure.
3. **Test your changes** locally by reloading the unpacked extension in Chrome.
4. **Submit a Pull Request** with a detailed explanation of your changes.

## Bug Reports & Feature Requests
Please use the GitHub Issue tracker to report bugs or request features. When reporting a bug, include your Chrome version and steps to reproduce.
